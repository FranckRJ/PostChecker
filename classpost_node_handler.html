◆ getNumberOfNodesWithoutResult()

int postNodeHandler::getNumberOfNodesWithoutResult ( ) const

Retourne le nombre de node arrêtées sans résultat.

Une node sans résultat est une node dont la différence entre les éléments de la paire est telle qu'il est impossible de lui ajouter une autre paire. La différence entre les éléments de la paire n'est pas nulle sur cette node.

Returns
Le nombre de node arrêtées sans résultat.
